# 📊 데이터마이닝과분석 텀프로젝트 (회귀 · 정규화)

## 🧭 프로젝트 개요

본 프로젝트는 **회귀 예측 모델 구축**을 목표로 한 데이터마이닝과분석 과목 텀프로젝트입니다.  
교수님 제공 단일 데이터셋을 기반으로, 팀 단위로 학습·검증 데이터를 분할하고 **선형회귀 계열 모델만 사용**하여 분석을 수행했습니다.

- **주제**: 수치형 변수 기반 회귀 예측
    
- **데이터**: 교수 제공 단일 파일
    
- **팀 구성**: 3~4인
    
- **Seed 고정**: `random_state = 42`
    

---

## 🎯 분석 목표

- 비선형 모델을 배제하고, **해석 가능한 선형 모델**의 성능과 한계를 명확히 이해
    
- 전처리, 변수 선택, 정규화가 회귀 성능과 일반화에 미치는 영향 분석
    
- Validation 성능과 Test 성능 간 괴리를 최소화하여 **과적합 방지**

## 📈 평가 지표 및 최종 성능 점수

본 프로젝트의 성능 평가는 **가산점 전용 지표**로 활용되었으며, 아래의 최종 성능 점수 **M**을 기준으로 순위를 산정했습니다.

### 사용 지표

- **Adjusted R²** (수정된 결정계수)
    
- **MAPE** (Mean Absolute Percentage Error)
    

### 최종 성능 점수 (M)

```
M = ( max(0, Adjusted R²) + max(0, 1 − MAPE) ) / 2
```

- Adjusted R²는 변수 수(p)와 표본 수(n)를 고려하여 계산
    
- MAPE는 소수 단위로 계산 (예: 15% → 0.15)
    
- **Validation 성능과 Test 성능 간 괴리가 클 경우 과적합으로 판단되어 감점**
